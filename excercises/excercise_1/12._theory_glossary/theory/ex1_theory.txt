a) WHAT ARE THE MAIN CATEGORIES OF SQL COMMANDS?* Data Query Language (DQL): Används för att fråga och hämta data från databasen.  -- SELECT  * Data Manipulation Language (DML): För att hantera data i tabeller:  -- INSERT  -- UPDATE  -- DELETE* Data Definition Language (DDL): För att definiera och ändra databasens struktur:  -- CREATE  -- ALETER  -- DROP  * Data Control Language (DCL): För att hantera åtkomst till databasen:  -- GRANT  -- REVOKE* Transaction Control Language (TCL): För att hantera transaktioner:  -- COMMIT  -- ROLLBACK  -- SAVEPOINTb) Explain the difference between HAVING and WHERE clauses?* WHERE: Används för att filtrera rader innan gruppering sker i en fråga.* HAVING: Används för att filtrera grupper efter gruppering.c) How to make sure you delete the correct rows?* Granska med SELECT: Kör först en SELECT-fråga med samma villkor som din DELETE för att verifiera vilka rader som påverkas.* Använd WHERE-klausul: Inkludera ett tydligt och specifikt villkor.* Använd en transaktion: Om databasen stöder transaktioner, kan du använda BEGIN TRANSACTION och granska innan COMMIT.d) How do you retrieve unique values in a column?* Använd nyckelordet DISTINCT i din SELECT-satse) What does data transformation mean?f) How do you create a new row in a table?* Använd INSERT INTO-kommandot:g) What happens if you omit the WHERE clause in an UPDATE statement?h) What happens if you omit the WHERE clause in a DELETE statement?i) What is a conditional statement in SQL, and can it be used with SELECT?
